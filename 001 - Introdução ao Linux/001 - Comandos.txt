Comandos Linux
1. Informar todos os IPs disponiveis no computador
ip a
2. Verificar enp0s3 e então inet.

Conectando na maquina virtual Linux local via Windows
1. Realizar o download do programa Putty para realizar o acesso remoto com a maquina virtual
2. No Putty, informar o ipv4 da maquina linux e conectar
3. Caso o Putty rejeite a conexão, será necessário baixar na maquina virtual um serviço openssh
4. Digite "apt-get install openssh-server". Caso retorne nao autorizado, digite "sudo" no começo do comando para fazer acesso de adm


Conectando na maquina virtual Linux AWS via Windows
1. Realizar o download do programa Putty para realizar o acesso remoto com a maquina virtual
2. Realizar o download do Puttygen.exe para gerar chaves
3. No Puttygen, selecionar Load, procurar o arquivo .pem (Gerado na hora de criar a maquina virtual AWS) para gerar uma chave
4. Clicar em "Save private key" e salvar o arquivo .ppk gerado
5. Opcionalmente pode informar uma senha para a chave
6. Agora no software Putty, selecione "Connection > SSH > Auth > Credential" e então, vá em browser para selecionar o arquivo .ppk gerado
7. Selecionar a aba Session e informar o IP da maquina virtual AWS e clicar em Open
8. O AWS gera um usuario padrão chamado ubuntu
